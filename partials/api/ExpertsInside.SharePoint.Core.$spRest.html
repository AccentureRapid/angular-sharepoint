$spRest
service in module ExpertsInside.SharePoint.Core

Description

Utility functions when interacting with the SharePoint REST API

Dependencies

  • $log

Methods

  • appendQueryParameters(url, params)

    Builds a query string from the query parameters and appends it to the url

    Parameters
    ParamTypeDetails
    urlstring

    url

    paramsObject

    query parameters

    Returns
    string

    url with query string

    Example

    var params= {
  $select: ['Id', 'Title']
};
url = $spRest.appendQueryParameters('http://my.app', params);
// url => "http://my.app?$select='Id,Title'"

  • buildQueryString(params)

    Create a query string from query parameters that SharePoint accepts

    Parameters
    ParamTypeDetails
    paramsObject

    query parameters

    Returns
    string

    query string

    Example

    var params= {
  foo: [1,2,3]
  bar: "baz"
};
var qs = $spRest.buildQueryString(params);
// qs => 'foo="1,2,3"&bar="baz"'

  • normalizeParams(params, whitelist)

    Normalizes the query parameters by prefixing them with prefixing them with $ (when missing) and removing all invalid query parameters when a whitelist is given.

    Parameters
    ParamTypeDetails
    paramsObject

    query parameters

    whitelistArray.<string>

    allowed query parameters

    Returns
    Object

    normalized query parameters

    Example

    var params = {
  select: ['Id', 'Title']
  invalid: "foo"
};
var whitelist = ['$select']
params = $spRest.normalizeParams(params);
// params => { $select: ['Id', 'Title'] }

  • transformResponse(json)

    Parse the JSON body and remove the d and d.results wrapper from the REST response

    Parameters
    ParamTypeDetails
    jsonstring

    JSON body of the response

    Returns
    Object|Array

    transformed response

    Example

    var json='{"d":{"results":[{"foo":"bar"}]}}';
var response = $spRest.transformResponse(json);
// response => [{ foo: "bar" }]
