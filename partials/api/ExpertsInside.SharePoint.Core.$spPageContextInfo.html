$spPageContextInfo
service in module ExpertsInside.SharePoint.Core

Description

A reference to the documents _spPageContextInfo object. While _spPageContextInfo is globally available in JavaScript, it causes testability problems, because it is a global variable. When referring to it thorugh the $spPageContextInfo service, it may be overridden, removed or mocked for testing.

See _spPageContextInfo Properties for more information.

Example

function Ctrl($scope, $spPageContextInfo) {
   $scope.userName = $spPageContextInfo.userLoginName
 }
